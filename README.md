# 🚀 ES6+ Practice
자바스크립트의 동작 원리를 깊이 있게 이해하고, 현대적인 자바스크립트(ES6+) 문법을 실무에 가깝게 익히기 위해 진행한 자기주도 학습 프로젝트입니다.

## 📌 Project Purpose (학습 목표)
자바스크립트의 기초가 부족하다는 점을 인지하고, 이를 보완하기 위해 다음과 같은 핵심 요소들을 직접 코드로 구현하며 학습했습니다.

- JS Engine 이해: 실행 컨텍스트, 클로저, 호이스팅 등 자바스크립트의 고유한 동작 방식 습득

- 현대적 문법 숙달: ES6(ECMAScript 2015) 이후 도입된 문법들을 활용한 효율적인 코드 작성

- 비동기 프로그래밍: Promise, async/await를 활용한 비동기 처리 흐름 파악

- 코드 가독성 향상: 더 간결하고 유지보수가 쉬운 코드 작성법 고민

## 🛠 Tech Stack
- Language: JavaScript (ES6+)

- Environment: Node.js / Browser Runtime

## 📂 Key Learning Modules (주요 학습 내용)
프로젝트 내부는 주제별로 세분화되어 관리되고 있으며, 각 파일은 특정 개념의 이론과 실습을 포함합니다.

### 1. Fundamental Concepts
- 변수 선언(var, let, const)의 차이점과 스코프 체인

- 자료형(Data Types)과 암묵적 타입 변환의 위험성

- 함수의 다양한 선언 방식과 this 바인딩

### 2. ES6+ Features
- Destructuring: 구조 분해 할당을 통한 데이터 추출 최적화

- Spread & Rest Operators: 배열과 객체의 효율적인 조작

- Template Literals: 가독성 높은 문자열 조합

- Modules: import/export를 활용한 코드의 모듈화

### 3. Asynchronous JavaScript
- 콜백 지옥(Callback Hell)의 해결 방안 탐구

- Promise 객체의 상태와 체이닝

- async/await를 통한 동기적인 코드 흐름 구현

### ✍️ Insights & Reflection (성찰 및 배운 점)
- 언어의 깊이: 단순히 API를 사용하는 수준에서 벗어나, 자바스크립트 엔진이 코드를 어떻게 해석하는지 공부하며 디버깅 능력이 크게 향상되었습니다.

- 지속 가능한 학습: 매일 새로운 개념을 코드로 직접 타이핑하고 테스트하며, 이론과 실습의 간극을 줄이는 과정을 경험했습니다.

- 발전 방향: 현재는 기초 문법 위주이지만, 앞으로는 이를 활용한 알고리즘 풀이 및 프레임워크(React 등)로의 확장을 목표로 하고 있습니다.

### 🏃 How to Use
각 파일은 독립적으로 실행 가능합니다.

```Bash
# 레포지토리 클론
git clone https://github.com/rsl150101/es.git

# 해당 디렉토리로 이동
cd es

# Node.js 환경에서 특정 파일 실행 예시
node [파일명].js
```
